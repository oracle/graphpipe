# Client Installation

::: tabs install

- python

    ```bash
    git clone https://github.com/oracle/graphpipe-py
    cd graphpipe-py
    python setup.py install
    ```

- golang

    ```bash
    go get github.com/oracle/graphpipe-go
    cd $GOPATH/src/github.com/oracle/graphpipe-go
    make install-govendor
    make deps
    ```
